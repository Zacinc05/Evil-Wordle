Wordle Roguelike.
Command line Wordle game with a Bot that Gaslights you.

Game Rules: You Start with 5 guesses.
A bot will suggest words every other turn.
Depending on your level of trust, the bot may begin to lie more.
Enter a 5 letter word.
After each Wordle, a shop will offer upgrades
Goal: Survive 10 rounds

How to play:
Simple python file. Make sure Python is installed on PC.
Word list should be in the same directory as the game file.
Open command line
Download the game, and cd to the directory you placed it in.
Run 'py game.py' in command line

Alternatively you can run the python file 'game.py' however else.
